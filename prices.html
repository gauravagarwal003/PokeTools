This page shows the current and historical data for these three values:

Expected Value (EV): The expected value of a pack, calculated by weighting each card's market value by its chance of being pulled.

Pack Price: The current market price of a pack.

Percent Return: How much you'd expect to recover in value if you bought the pack at market price.

These values are updated daily. Note: Shrouded Fable is not included as TCGplayer Infinite did not release pull rates for it.
